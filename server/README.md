# FunTodo Server

Node.js Express server for the FunTodo client application.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Environment variables are configured in `.env`

3. Run development server:

```bash
npm run dev
```

4. Start production server:

```bash
npm start
```

## API Endpoints

- `GET /` - Welcome message
- `GET /api/health` - Health check

## Technologies

- Express.js
- Prisma ORM
- PostgreSQL
- OpenAI API
